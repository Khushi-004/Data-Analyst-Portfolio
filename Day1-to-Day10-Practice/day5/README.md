# Day 5 - Sales Data Analysis & Interactive Dashboard

## 📋 Task Overview
Created an interactive Excel dashboard for sales data analysis. Cleaned raw data, fixed date anomalies, and built visualizations with slicers for dynamic filtering.

## 🔧 Tools Used
- **Microsoft Excel**: Power Query, Pivot Tables, Pivot Charts, Slicers
- **Git/GitHub**: Version control

## 📊 Key Steps Performed

### 1. Data Cleaning
- Imported raw `sales_data.csv` into Excel
- Used **Power Query** to fix incorrect year `2039` → `2024` in date column
- Removed null values and standardized column formats

### 2. Data Analysis
- Created **Pivot Table** to analyze sales by:
    - Region-wise performance
    - Product category breakdown  
    - Monthly sales trends

### 3. Dashboard Creation
- Built **Pivot Chart**: Column chart showing Total Sales by Region
- Added **Slicers** for interactive filtering:
    - Date/Month
    - Region 
    - Product Category
- Formatted dashboard for better readability

## 📁 Files in this Folder
- `Task3_Sales_Dashboard.xlsx` - Final Excel file with cleaned data, pivot table, chart, and slicers
- `sales_data.csv` - Raw dataset used

## ✅ Key Learning
- Power Query for advanced data cleaning
- Dynamic dashboards using Pivot + Slicers
- Handling date/time anomalies in real-world data
